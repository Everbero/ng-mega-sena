Sobre este componente

Este componente de login aceita qualquer usuário e senha e se comporta desta forma:


  1. Se não houver um usuário, um novo usuário é cadastrados com os dados informados;
  2. Se já existir um usuário, a senha é verificada contra o que estiver salvo
  3. Todos os dados e jogos são salvos no localStorage com o prefixo do usuário, permitindo a persistência de dados para vários usuários
