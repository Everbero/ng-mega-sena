Sobre este projeto

NG megasena é um projeto angular simples feito para um trabalho de faculade. Este projeto contém 3 componentes (app, dashboard e home), 2 serviços que consomem as APIs localStorage e uma api genérica que retorna o resultado do último sorteio da megasena.

Observações

Infelizmente a API que eu iria utilizar está fora do ar: guto alves loterias então estou usando um link provisório: api caixa, esta não é uma API oficial, não aceita aceita CORS então estou usando um proxi para acessar os dados, se os dados não estiverem disponíveis, tente novamente em alguns minutos.

Pretendo extender este projeto futuramente durante o curso adicionando a capacidade de detectar se um dos jogos gerados foi sorteado, isso caso o autor da api volte a dar manutenção.
